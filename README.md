# Introduction to Programming - Wordle
## Usage
- Use G++ to compile the program.
- Once the program is open, you will be prompted with a menu consisting of 3 options:
    - Login
    - Register
    - Exit
- You can login as either a user or an admin.
    - Users can play the game and accumulate scores locally.
    - Admin accounts can add/remove words and view the leaderboard.
- The goal of the game is to guess an unknown, 5-letter word within 6 attempts.
- All results are saved within the .txt files locally.
